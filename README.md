
# How to run application
1. Clone the repository
```bash
git clone <repo-url>
```

2. Change directory to the cloned repository
```bash
cd <repo-name>
```

3. Install dependencies
```bash
yarn install
```
or 
```bash
npm install
```

4. mmigrate database
```bash
yarn migrate:up
```

5. Run the application
```bash
yarn start
```
